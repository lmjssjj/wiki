## Shortcut key

```
ctrl+alt+t:开启终端
alt+f4:关闭终端
ctrl+l:清屏
Ctrl + Alt + (F1~F6中的任意一个)即可进入纯命令行模式
Ctrl + Alt + F7即进入图形界面模式。
shift + PgUp + PgDn  上下翻页
```

## ls

```
当前目录下的文件
ls -a[all] ： 显示所有文件包括隐藏文件(.开头的文件或文件名)
ls 目录名 ： 查看此目录下文件
```

## rm

```

```
